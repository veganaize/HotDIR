# HotDIR

Colorized directory listings for the Windows command prompt.


## Installation

Place the `hd.exe` executable somewhere in your `PATH`.


## Usage

```cmd
C:\> hd
C:\> hd *.exe
C:\> hd x:\some\path
C:\> hd x:\some\path\*.txt
```
